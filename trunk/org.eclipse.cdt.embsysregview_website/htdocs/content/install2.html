Install the view by copying the jar files:

Get the latest jar files (you need both of them) from https://sourceforge.net/projects/embsysregview/files/

Simply copying to your eclipse plugins folder, and your done.

Optional:

If you wish to manipulate the xml files or add your own, extract the file org.eclipse.cdt.embsysregview_x.x.x.jar into a directory called "org.eclipse.cdt.embsysregview_x.x.x" in your plugin folder (using the update site, this is done automaticaly). Now you have access to the data folder containing the xml files.
