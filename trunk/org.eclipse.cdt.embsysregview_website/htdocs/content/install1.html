Install the view using the Update-Site:

First of all, get yourself a copy of "Eclipse IDE for C/C++ Developers" from eclipse.org

For Eclipse versions >= Keppler you can search for "embsysregview" in the eclipse Marketplace (Help->Eclipse Marketplace...)

or use the Stable update site URL: http://embsysregview.sourceforge.net/update

Beta update site URL: http://embsysregview.sourceforge.net/update-beta
