Howto setup a development environment

  • Install Eclipse SDK
  • add CDT
  • add CDT device debug (C/C++ GDB Hardware debugging)
  • add subclipse
  • checkout source (org.eclipse.cdt.embsysregview)
  • select "org.eclipse.cdt.embsysregview" in Package Explorer
  • click the run button to launch it as Eclipse Application
  • or set a breakpoint and click the debug button
  • a new Eclipse with the running plugin will open (you have to show it the first time ...Window->Show View->Others...)
  • in the Run Configuration you can select a workspace (probably the one that contains your embedded project) that the launched eclipse will use
